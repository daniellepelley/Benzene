# Benzene.Tools

## What this package does
Command-line tools and utilities for Benzene development. Provides CLI commands for scaffolding, code generation, project analysis, and development workflows.

## Key types/interfaces

### Development Tools
- CLI scaffolding commands
- Project analysis tools
- Development utilities

## When to use this package
- During development for scaffolding
- For generating boilerplate code
- When analyzing Benzene projects
- For development workflow automation

## Dependencies on other Benzene packages
- Various Benzene packages for analysis

## Important conventions
- Installed as global or local tool
- Commands follow dotnet tool conventions
- Used during development, not runtime
