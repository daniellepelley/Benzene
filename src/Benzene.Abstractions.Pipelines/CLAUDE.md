# Benzene.Abstractions.Pipelines

## What this package does
Pipeline abstractions for Benzene. Provides higher-level pipeline interfaces beyond basic middleware, enabling complex pipeline composition and branching patterns.

## Key types/interfaces

### Pipeline Abstractions
- Advanced pipeline interfaces
- Pipeline branching abstractions
- Pipeline composition patterns

## When to use this package
- When building complex pipeline architectures
- For advanced middleware composition
- Rarely used directly - consumed by Core packages

## Dependencies on other Benzene packages
- **Benzene.Abstractions** - Core abstractions
- **Benzene.Abstractions.Middleware** - Middleware abstractions

## Important conventions
- Higher-level than basic middleware
- Enables pipeline composition patterns
- Used by pipeline-heavy scenarios
