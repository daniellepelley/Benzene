## Multi cloud

Run Benzene services on Azure Functions, AWS Lambdas or Google Functions. Effortlessly swap between different cloud providers.

Run the same service on any of the following.

- Amazon Web Services
  - Lambda
  - EKS
  - Elastic Beanstalk
- Azure
  - Function
  - Web Service
  - AKS
- Google Cloud
  - Function
- Kubernetes
- IIS
