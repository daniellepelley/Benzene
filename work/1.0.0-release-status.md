**Ship at 1.0.0:**

- Benzene.Abstractions.Middleware
- Benzene.Core
- Benzene.Core.Middleware
- Benzene.Http

**Keep at 0.9.x-preview:**
- Benzene.Abstractions.MessageHandlers (needs API surface reduction)

- Benzene.Aws.Lambda.ApiGateway (has test helpers)

- Gives users a solid, stable foundation
- Allows more time to mature message handling layer
- AWS adapters can stabilize independently
- Reduces scope and risk for 1.0
